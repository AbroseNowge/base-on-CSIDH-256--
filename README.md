# base-on-CSIDH-256--
基于后量子密码CSIDH-256的优化

采用Montgomery算法框架，针对特殊素数p设计定制化模约简路径；使用__uint128_t类型实现硬件级64×64乘法加速；通过循环展开、内存对齐、编译器优化（-O3、LTO、PGO）及无分支实现提升指令级并行与缓存命中率；构建三层测试体系验证性能、正确性与稳定性；独立开发交互演示系统，实现算法调用、性能对比与结果可视化。

主要可交互运行程序为：**demo_windows.exe 、 interactive_key_exchange.exe**
具体运行方式可查看文件夹内readme.md进行环境配置
